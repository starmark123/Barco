Please run the test under selenium environment.

Open the terminal and change directory to Barco, then run the below command
	"pytest test_sample.py"
The test should be running successfully.

Based on different browser, you may need to download different browser driver for performing the test.
In this case, I am using ChromeDriver 103.0.5060.134.
You can download the chrome driver at https://chromedriver.chromium.org/downloads
